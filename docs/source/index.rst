Welcome to sc_intNMF's documentation!
=====================================


**sc_intNMF** is a python library for analysis of single cell multiomic data. Example data found at `10x genomics <https://www.10xgenomics.com/products/single-cell-multiome-atac-plus-gene-expression>`_. It offers a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, inculding how to :ref:`install <installation>` the project.

.. note::

    This project is under development.
